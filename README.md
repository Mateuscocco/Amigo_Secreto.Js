Este é um projeto desenvolvido para praticar lógica de programação.  
O sistema realiza o sorteio de nomes entre amigos para um Amigo Secreto, garantindo que:
- Nenhum participante tire o próprio nome;
- Cada pessoa receba apenas um nome.

---

## Funcionalidades

- Cadastro de participantes  
- Sorteio automático dos pares  
- Verificação para evitar que alguém tire o próprio nome  
- Exibição do resultado (no console, interface ou arquivo)

---

## Tecnologias utilizadas

- Linguagem: Python (ou JavaScript, Java, etc. — ajuste conforme o projeto)  
- Biblioteca: random (para gerar o sorteio)  
- Ferramentas opcionais: interface simples em terminal ou navegador

---

## Objetivo de aprendizado

Este projeto foi criado para praticar:
- Estruturas de repetição e condição  
- Manipulação de listas ou arrays  
- Geração de números aleatórios e sorteios  
- Entrada e saída de dados

---

## Como executar
- Baixe o projeto
- Abra o arquivo index.html no navegador
